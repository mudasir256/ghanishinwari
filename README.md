# Ghani Shinwari Restaurant

Website for **Ghani Shinwari Restaurant** — authentic Shinwari cuisine in Lahore (Truck Adda, Bedian Road, Bahria Town).

## Stack

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS 4**

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build for production

```bash
npm run build
npm start
```

## Contents

- **Menu**: Karahi, Kabab, Barbeque, Rosh, Pulao, Qeema, Special (Khada Dumba), Roti, Kehwa, Drinks, Extras — with English/Urdu names and prices from your menu images.
- **Locations**: All three Lahore branches with addresses.
- **Design**: Maroon, gold, and cream theme; script headings; Urdu script support; responsive layout.
